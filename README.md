已支持主流平台 但此方案需要一定的电视或盒子性能 版本最好安卓9及其以上

## 已实现功能
1.  央视频各大卫视打开全屏和各个频道切换
2.  CCTV和各个地方台适配到电视端
3. 爱奇艺 腾讯视频 哔哩哔哩 优酷 芒果适配到电视端
4.  进入视频详情页 如果平台位登录 跳出登录二维码
5.  进入视频详情页 可以全屏播放视频 可以遥控器选集 进度 倍数等操作
